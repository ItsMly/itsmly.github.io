So aktualisierst du die Firmware auf der Samsung PM983 #

Samsung ist berüchtigt dafür, keine Firmware für ihre Enterprise-SSDs herauszubringen. Aber jetzt ist es soweit - los geht’s mit dem Update!

Downloads #

Beispiel PM983 #

  • PM983 U.2 Firmware beginnt mit EDAxxxxx z.B. EDA5702Q

  • (SAMSUNG MZ1LB3T8HMLA-00007) PM983 M.2 Firmware beginnt mit EDBxxxxx z.B. EDB7602Q

Flashe nicht die falsche Firmware!

Modellcodes für Samsung PM983 #

  • M.2 Modelle:

    • SAMSUNG MZ1LB3T8HMLA-00007 (3,84 TB)
    • SAMSUNG MZ1LB1T9HALS-00007 (1,92 TB)
    • SAMSUNG MZ1LB960HAJQ-00007 (960 GB)

  • U.2 Modelle:

    • SAMSUNG MZQLB7T6HMLA-00007 (7,68 TB)
    • SAMSUNG MZQLB3T8HALS-00007 (3,84 TB)
    • SAMSUNG MZQLB1T9HAJR-00007 (1,92 TB)
    • SAMSUNG MZQLB960HAJR-00007 (960 GB)

Die richtige SSD identifizieren #

$ nvme list

--------------------- --------------------- -------------------- ---------------------------------------- --------- -------------------------- ---------------- --------

Wichtige Infos #

Aus der Ausgabe des nvme id-ctrl Befehls können wir alle benötigten Informationen entnehmen. Zum Beispiel:

mn        : SAMSUNG MZ1LB3T8HMLA-00007
fr        : EDB7602Q
...
frmw      : 0x17
  [5:5] : 0     Multiple FW or Boot Update Detection Not Supported
  [4:4] : 0x1   Firmware Activate Without Reset Supported
  [3:1] : 0x3   Number of Firmware Slots
  [0:0] : 0x1   Firmware Slot 1 Read-Only

Aus der Ausgabe des nvme id-ctrl Befehls können wir die Modellnummer und Firmware-Informationen entnehmen.

[3:1] : 0x3 => 3 Firmware-Slots
[0:0] : 0x1 => Read-Only Firmware-Slot - NICHT HIER FLASHEN

Zuerst müssen wir die Firmware-Datei in den Controller-SRAM senden:

nvme fw-download -f EDB7602Q.bin /dev/nvme0
Firmware download success

Dieser Befehl lädt die Firmware-Datei in den Speicher der SSD. Als nächstes flashen wir die heruntergeladene Firmware in Slot 2, ohne sie zu aktivieren:

nvme fw-commit -s 2 -a 0 /dev/nvme0
Success committing firmware action:0 slot:2

-s 2 = Slot 2
-a 0 = Nicht aktivieren

Nach dem erfolgreichen Flashen aktivieren wir die Firmware:

nvme fw-commit -s 2 -a 2 /dev/nvme0
Success committing firmware action:2 slot:2

Herzlichen Glückwunsch, du hast die Firmware aktualisiert! Jetzt herunterfahren und die SSD neu starten und hoffen, dass alles klappt :)

Wie man Sektoren auf der Samsung PM983 ausrichtet #

Die PM983 kann Sektorgrößen von 4k oder 512 unterstützen. So schaltest du auf 4k Sektorgröße um. Um die aktuelle Sektorgröße zu überprüfen, benutze den Befehl nvme id-ns:

$ nvme id-ns /dev/nvme0n1
...
lbaf  0 : ms:0   lbads:9  rp:0
lbaf  1 : ms:0   lbads:12 rp:0 (in use)

Achte auf die lbaf (Logical Block Addressing Format) Einträge: lbaf 0: 512 Bytes Sektorgröße (lbads:9) lbaf 1: 4k Sektorgröße (lbads:12)

Um die Sektorgröße zu ändern, benutze den nvme format Befehl. Achtung: Dies wird das Laufwerk formatieren! Sichere deine Daten.

Achtung: Dies formatiert das Laufwerk! Sichere deine Daten.
nvme format /dev/nvme0n1 --lbaf=1 --reset

lbaf=x wobei x der Slot mit dem lbads:y ist, den wir möchten.

Jetzt haben wir die Sektorgröße auf 4k Sektoren umgestellt. :)
